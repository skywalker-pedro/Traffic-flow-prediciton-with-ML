\slide{Resultados obtidos}{
\begin{enumerate}
    \item Agentes foram capazes de aprender a partir da observação dos outros jogadores.
    \item Para agente Homogêneos, os agentes treinados tiveram comportamento similar ao modelo de referência.
    \item Para agentes Heterogêneos, o agente foi capaz de melhorar seu desempenho quando o custo do atraso individual era consideravelmente maior que o do antecessor. 
    \item Baixo número de estados de negociação. 
\end{enumerate}
}
